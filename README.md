Using a dataset containing MRI and CT scan images from kaggle, I developed two projects : 


- In multimodal_classification folder, I am trying to compare simple segmentation with multimodal classification using both MRI and CT scan images

- In ImageGeneration, I am trying threw different types of models (diffusion, VAE, GAN) to generate images, and the aim is then to develop 
super-resolution model using VAE and diffusion methods on downsampled images.
